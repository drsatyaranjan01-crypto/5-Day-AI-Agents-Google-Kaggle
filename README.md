## 📌 Overview

This repository documents my learning journey through the **Google × Kaggle 5-Day AI Agents: Intensive Vibe Coding Course**.

The course focused on building modern AI Agents using Google's ecosystem, including **Gemini**, **Agent Development Kit (ADK)**, **Agents CLI**, **Model Context Protocol (MCP)**, **Cloud Run**, and production-grade deployment practices.

---

## 🎯 Course Objectives

- Understand AI Agents and Agentic Workflows
- Learn Vibe Coding with AI-assisted development
- Build and deploy AI Agents using Google ADK
- Explore MCP (Model Context Protocol)
- Develop Multi-Agent Systems
- Learn Agent Skills and Context Management
- Understand Agent Security & Evaluation
- Deploy Production-Grade AI Applications
